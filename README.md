## Will Blair

I build [Vela](https://github.com/constellate-science/vela), version control for scientific state. It's the core of Constellate, open infrastructure for cumulative science.

Vela turns a scientific result into content-addressed, verifier-gated state that anyone can reproduce and build on. The case for why that matters is in three essays, [Constellations of Borrowed Light](https://constellate.science).

By day I build researcher-intelligence tooling at an R&D lab in San Francisco.
